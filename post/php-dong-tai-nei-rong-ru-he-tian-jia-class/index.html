PHP动态内容，如何添加Class？

· 2019-08-21 · # WordPress # PHP

遇到过一个奇怪的需求，客户的邮箱是通过后台填写的，然后前台获取，获取的是一串字符串，例如:xxx@mail.com
但是客户需求是@这个字体要跟别的不一样。。。我只能通过添加class的方法，解决这个需求。

首先 获取动态内容、根据"@“分组字符串。

 $email = explode('@',get_field('email', 90));

然后串联添加输出即可

echo $email[0].'<span class="email-ico">@</span>'.$email[1];
